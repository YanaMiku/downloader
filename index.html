Informasi CORS

Instagram membatasi akses langsung ke file media untuk mencegah hotlinking. Untuk mengunduh gambar, Anda dapat:

  1. Klik kanan pada gambar preview di atas dan pilih "Save image as..."
  2. Gunakan tombol download di atas (membutuhkan klik ekstra)
  3. Untuk video, download akan bekerja secara normal

Download Manual

Jika tombol download tidak berfungsi:

  1. Klik kanan pada gambar di preview
  2. Pilih "Save image as..." atau "Simpan gambar sebagai..."
  3. Simpan dengan format .jpg
